## ✅ Закрытые вопросы (с зафиксированными решениями)

| ID    | Вопрос                                                                                                | Решение                                                                                                                                                                                           | Статус     |
| ----- | ----------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- |
| V-C01 | Какой именно платёжный шлюз будет интегрирован?                                                       | **Решение:** ЮKassa (основной), CloudPayments (резервный). Обоснование: соответствие PCI DSS Level 1, поддержка Apple Pay/Google Pay, комиссия ~3.5%, наличие готовых SDK для iOS и Android.      | ✅ Закрыто |
| V-C02 | Алгоритм подтверждения возврата при неточной геолокации?                                              | **Решение:** Радиус 20 метров + временное окно 3 минуты. Если GPS не подтверждает — клиент нажимает "Завершить вручную" + фото велосипеда. Оператор проверяет фото (SLA: ≤ 15 мин).               | ✅ Закрыто |
| V-C03 | Как обрабатывать ситуации, когда велосипед не вернули до 23:00?                                       | **Решение:** Автозавершение + штраф 500 руб. Push-уведомление за 30 минут до автозавершения. Если велосипед не на станции — вызов курьера (SLA: 1 час).                                           | ✅ Закрыто |
| V-C04 | Нужна ли интеграция с Apple Pay/Google Pay в MVP?                                                     | **Решение:** Да, через ЮKassa (поддерживается из коробки). Дополнительной интеграции не требуется.                                                                                                | ✅ Закрыто |
| V-C05 | Входит ли в бюджет 2 млн рублей серверная часть, интеграции, админ-панель, тестирование и публикация? | **Решение:** Да, бюджет включает все статьи. Декомпозиция: мобильное приложение (iOS + Android) — 1.2 млн, бэкенд + API — 400 тыс., админ-панель — 200 тыс., тестирование и публикация — 200 тыс. | ✅ Закрыто |
| V-C06 | Каков объём админ-панели в MVP?                                                                       | **Решение:** MVP включает только функции оператора (просмотр тикетов, возвраты, ручное завершение аренд). Полная админ-панель (управление парком, тарифами, аналитика, промокоды) — в версии 2.0. | ✅ Закрыто |
